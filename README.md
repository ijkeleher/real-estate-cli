# RealEstate-CLI
Simple CLI app built with Java. Has list, buy, sale, save current sales data and summary to .txt and an auction bidding feature.

## To run app
1. git clone 
2. javac *.java
3. java RealEstateSystem

## App functions
  * Add New Sale	**A**
  * Submit Offer __B__
  * Display Sales Summary __C__
  * Add New Auction __D__
  * Close Auction __E__
  * Exit Program	__X__
